# 🏋️‍♂️ GymPro: Integrated Gym Management System

### **Project Overview**
GymPro is a full-stack digital solution designed to modernize gym operations and enhance member engagement. This project was developed as a collaborative software engineering capstone, focusing on a seamless connection between a cross-platform mobile app and a robust administrative backend.

### **System Architecture**
The project is structured into two main modules to ensure high professional standards and clear separation of concerns:
* **`mobile_app/`**: A high-performance mobile application built with **Flutter**, providing members with tools to manage subscriptions and track their workout journey.
* **`backend_api/`**: A secure and scalable administrative backend powered by **Laravel**, handling API endpoints, business logic, and database synchronization.

### **Key Features**
* **Member Management:** Comprehensive dashboard for tracking membership status and workout history.
* **Class & Schedule Management:** Integrated tools for owners to manage gym sessions and attendance via the Laravel dashboard.
* **Real-time Synchronization:** Ensures consistent data flow between the mobile client and the central MySQL database.
* **Security Focused:** Implements industry-standard practices, including environment variable management for API keys.
