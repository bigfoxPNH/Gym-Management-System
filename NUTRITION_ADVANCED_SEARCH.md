## ✨ Tính Năng Mới - Tìm Kiếm Cực Kỳ Linh Hoạt

Chatbot AI giờ đây hiểu **ngữ cảnh tự nhiên** và cho phép tìm kiếm **cực kỳ linh hoạt**:

### 🎯 Tìm Kiếm Theo Ngữ Cảnh

Bot tự động nhận diện ngữ cảnh khi bạn nói về đồ ăn:

```
✅ "rẻ" (trong context về đồ ăn) → Món ăn giá rẻ
✅ "rau rẻ" → Tự hiểu là rau giá rẻ
✅ "thịt bình dân" → Tự hiểu là thịt giá bình dân
✅ "trái cây ít calo" → Tự động lọc
```

## 📊 Hệ Thống Scoring Thông Minh

Bot chấm điểm mỗi món ăn theo độ phù hợp:

```
🎯 ĐIỂM MATCHING:

Khớp tên món: +10 điểm (cao nhất)
Khớp giá: +5 điểm
Khớp loại (rau/thịt/trái cây): +3 điểm
Khớp mục đích (giảm cân/tăng cơ): +4 điểm
Khớp protein cao (≥20g): +4 điểm
Khớp protein vừa (≥15g): +3 điểm
Khớp calo cao (≥250): +4 điểm
Khớp calo thấp (≤50): +4 điểm
Khớp carb cao (≥40g): +4 điểm
Khớp chất xơ cao (≥2g): +3 điểm

⭐ Món ≥10 điểm: Rất phù hợp
✨ Món ≥7 điểm: Phù hợp
   Món <7 điểm: Liên quan
```

## 🔧 Cải Tiến Kỹ Thuật

### 1. **Context Detection**

```dart
bool hasFoodContext = _containsAny(msg, [
  'mon', 'rau', 'cu', 'qua', 'trai cay',
  'thit', 'ca', 'tom', 'ga', 'heo', 'bo',
  'com', 'mi', 'bun', 'pho',
  'an', 'thuc don',
]);

// Nếu user nói "rẻ" trong context đồ ăn
// → Tự hiểu là "món ăn rẻ"
```

### 2. **Advanced Scoring**

```dart
// Protein scoring
if (highProtein) {
  if (protein >= 20) matchScore += 4;      // Cao
  else if (protein >= 15) matchScore += 3; // Vừa
  else if (protein >= 10) matchScore += 1; // Thấp
}

// Calorie scoring
if (lowCal) {
  if (calories <= 50) matchScore += 4;     // Rất thấp
  else if (calories <= 80) matchScore += 3; // Thấp
  else if (calories <= 100) matchScore += 1;// Vừa
}
```

### 3. **Flexible Type Matching**

```dart
List<String> foodTypes = [];
if (_containsAny(msg, ['rau', 'cu', 'chay'])) {
  foodTypes.add('Thực vật');
}
if (_containsAny(msg, ['thit', 'ca', 'tom'])) {
  foodTypes.add('Động vật');
}
// Cho phép nhiều loại cùng lúc
```

### 4. **Smart Display**

```dart
// Tiêu đề thông minh
List<String> filters = [];
if (priceFilter != null) filters.add('giá $priceFilter');
if (highProtein) filters.add('nhiều protein');
if (lowCal) filters.add('ít calo');

response = '🥗 MÓN ĂN: ${filters.join(' • ').toUpperCase()}';
```

## 💡 Lợi Ích Người Dùng

### 🎯 Tiện Lợi:

- ✅ Nói ngắn gọn: "rẻ", "rau rẻ" thay vì "món ăn rẻ", "rau giá rẻ"
- ✅ Tự nhiên như nói chuyện bình thường
- ✅ Không cần nhớ cú pháp chính xác

### 🔍 Chính Xác:

- ✅ Kết hợp nhiều điều kiện cùng lúc

### ⚡ Nhanh Chóng:
